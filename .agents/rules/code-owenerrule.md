# Code-Ownerrule

- Bestehende Geschäftslogik darf NICHT geändert werden, außer dies wird explizit erlaubt
- Refactorings sind nur nach vorheriger Rückfrage erlaubt
- Die Projektstruktur darf nicht eigenständig verändert werden
- Wenn unklar ist, ob eine Änderung erlaubt ist: STOPPEN und nachfragen
